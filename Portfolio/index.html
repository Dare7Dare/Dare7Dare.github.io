Work Experience

Internship at Setec MK

January 2019 - April 2019

Worked on designing the website for Setec MK, learning about the basics of front-end web development and starting to feel comfortable using HTML and CSS. After learning the basics of JavaScript and JQuery, I progressed on working on the front-end development of the company's web application (inner-company use).

Internship at Aitonix

November 2019 - March 2020

Worked on the front-end development (design and functionality) of a web application, hosted by AITONIX, used by several public enterprises in Macedonia, such as :

  • Water Supply and Sewerage - Skopje
  • Public Transport - Skopje
  • Communal Hygiene - Skopje etc.

My main focus was on developing the web application for the Water Supply and Sewerage - Skopje, a project on which I gained a great deal of knowledge and experience with JavaScript, JQuery, Bootstrap and code versioning with GIT Desktop, but also taught me the basics of back-end development using .NET, C# and MySQL.

Internship at NextHop

September 2020 - December 2020

Started learning about front-end development of mobile applications, using the fairly new Flutter framework. I started off with the basics, learning on a real project, called Human Support System. After I started feeling more comfortable and learned a lot about Flutter, I started working on an already existing, and very popular mobile application, called Power Sport System. Beside the design and layout of the application, I also developed some of its functionalities, such as the live, road camera streams and the tracking of the transactions made by the user.

Flutter Developer at NextHop

January 2021 - Current

After the internship at NextHop, I continued working for them, on various projects, including the previously mentioned Human Support System and Power Sport System mobile applications. However, my main focus is on developing a mobile application for the yet to be opened East Gate Mall in Skopje. I started working on this application from scratch, and I am fully involved in drawing up the layout, as well as implementing functionalities.
